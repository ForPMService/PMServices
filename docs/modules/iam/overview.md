# IAM Overview

## Функции

### Аутентификация
- Регистрация (личный / с организацией)
- Логин через Keycloak (OIDC)
- Password recovery
- Email verification

### Организации
- Создание
- Переключение между организациями
- Управление участниками

### RBAC
- Роли и permissions
- Назначение ролей
- Permission Discovery

### Приглашения
- В организацию (существующий пользователь)
- На платформу + в организацию (новый)

## Роли

| Роль | Описание |
|------|----------|
| Owner | Все права, нельзя удалить |
| Admin | Управление пользователями |
| Member | Базовая работа |

## Error Codes

| Code | Описание |
|------|----------|
| IAM-001 | Email уже зарегистрирован |
| IAM-002 | Невалидный invite token |
| IAM-003 | Организация не найдена |
| IAM-004 | Пользователь не найден |
| IAM-005 | Нет прав |
| IAM-006 | Rate limit |
| IAM-007 | Сессия истекла |
| IAM-008 | Аккаунт заблокирован |
| IAM-009 | Валидация |
| IAM-010 | Внутренняя ошибка |

## Rate Limiting

| Endpoint | Лимит |
|----------|-------|
| Registration | 5/min per IP |
| Forgot Password | 3/min per IP |
